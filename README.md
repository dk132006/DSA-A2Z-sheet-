# DSA A2Z Sheet

A structured repository containing solutions to the **A2Z DSA Sheet** in **C++**. This repository is designed to help students build a strong foundation in **Data Structures and Algorithms (DSA)** through consistent practice and well-organized code.

Whether you're preparing for coding interviews, competitive programming, or improving your problem-solving skills, this repository serves as a complete learning resource.

## How to Use

1. Clone the repository.

```bash
git clone https://github.com/your-username/DSA.git
```

2. Open the required topic folder.

3. Compile the C++ file.

```bash
g++ filename.cpp -o output
```

4. Run the program.

```bash
./output
```
